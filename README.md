# Bahamas Election Analytics Platform 2026

Real-time election analytics dashboard for the 2026 Bahamas General Election.

## Features
- Live constituency tracking
- Seat projections
- Interactive election map
- Snowflake data warehouse
- Real-time API ingestion
- Streamlit dashboard

## Tech Stack
- Python
- Snowflake
- Streamlit
- Plotly
- SQLAlchemy
- Pandas

## Architecture
API → ETL → Snowflake → MARTS → Streamlit Dashboard
